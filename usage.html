Usage

The following examples describe the basic usage of the Reproducible Maven plugin.

Fix build artifacts to make them reproducible

To execute the plugin during the packaging phase, add the Reproducible Maven plugin in the <build> section of your pom.xml.

<project>
  ...
  <build>
    <plugins>
      <plugin>
        <groupId>io.github.zlika</groupId>
        <artifactId>reproducible-maven-plugin</artifactId>
        <version>0.1</version>
        <executions>
          <execution>
            <phase>package</phase>
            <goals>
              <goal>strip</goal>
            </goals>
          </execution>
        </executions>
      </plugin>
    </plugins>
  </build>
  ...
</project>
